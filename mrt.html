Master Recruiting Techniques - Proposed Site Redesign

2015

Master Recruiting Techniques is a workshop specifically designed to help recruiters in the trucking industry learn how to use conversational recruiting techniques to pre-qualify and recruit drivers all at the same time. Their existing site had a single page layout where all of the information about the workshop was displayed. The result was a cluttered and confusing web page, so I proposed a multipage site redesign to better organize the info, make the call to action clearly visible, and incorporate more modern interface design layouts.
