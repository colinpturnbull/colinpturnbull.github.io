Master Recruiting Techniques - Site Redesign

2015

BACKGROUNDMaster Recruiting Techniques is a 2 day workshop that tours around the US and is specifically designed to help recruiters in the trucking industry learn how to use conversational techniques to pre-qualify and recruit drivers all at the same time. These courses provide recruiters access to the most comprehensive and industry specific recruiter training available.

OBJECTIVEAs an intern at Ramsey Mediaworks, I was given the task of collaborating with the in-house web developer on a redesign concept for the Master Recruiting Techniques site. The existing MRT website, where visitors would go to sign up for the workshop, used a single page layout. Using the existing website elements, I restructured the layout of the site, adding multiple pages to achieve a better visual hierarchy, flow, and a clear call to action while incorporating modern UI design elements.
